# Tito Ride Frontend

React frontend for the Tito Ride MTB Scheduler.

## Setup

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
```

## Technologies

- React 18
- React Router v6
- TailwindCSS
- Axios
- Vite
- date-fns
